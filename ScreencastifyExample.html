unique READme generator

Description:

  • This application generates a unique READme with the answers gathered by the Nodejs inquirer.
  • My motivation for creating this application was to simplify the READme creation process by creating an easy-to-use application that allows the user to answer a list of prompts and create a sufficient READme with skipping any sections.
  • While completing the development of the application, I learned how to use Nodejs in VS Code, as well as the inquirer function and html formatting in javascript to dynamically generate an HTML file with the desired information.
